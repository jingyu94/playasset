# PlayAsset AI 자산관리 기능맵 (2026)

## 1. 사용자 관점 핵심 기능 목록
1. 포트폴리오 정량 진단
- 샤프지수, 연환산 변동성, 최대낙폭(MDD), 최대보유비중, 분산점수 제공

2. 리밸런싱 액션 리스트
- 종목별 현재비중 vs 목표비중 비교
- 매수/매도 권장, 권장 금액, 우선순위, 사유 문장 제공

3. 추천 ETF 포트폴리오
- 사용자 리스크 상태와 집중도를 반영한 ETF 후보 추천
- 적합도 점수, 권장 비중, 총보수, 추천 사유 제공

4. LLM 스타일 인사이트 브리프
- 헤드라인 + 요약 + 핵심 포인트 + 주의사항
- 향후 실제 LLM 연동 시 프롬프트/근거 데이터 확장 가능 구조

## 2. 이번 구현에 반영한 DB 확장
1. `asset_classifications`
- 보유 종목의 카테고리/리스크 버킷 관리

2. `advisor_etf_catalog`
- ETF 추천 후보군 카탈로그
- 테마/리스크버킷/분산 역할/총보수 관리

3. `portfolio_advice_logs`
- 사용자별 진단 결과 생성 로그 적재

4. 시계열 데이터 보강
- 최근 90일 `market_price_candles` 샘플 데이터 보강
- 변동성/MDD/샤프 계산 안정화

## 3. 백엔드 아키텍처 반영
1. 신규 API
- `GET /api/v1/users/{userId}/portfolio/advice`

2. 응답 구조
- `metrics`: 정량 지표
- `rebalancingActions`: 리밸런싱 우선순위 목록
- `etfRecommendations`: ETF 추천 목록
- `insight`: AI 인사이트 문장형 결과

3. 계산 엔진 (규칙 기반)
- 수익률/변동성/MDD 기반 리스크 레벨 산정
- 목표 비중 편차 기반 리밸런싱 액션 생성
- 리스크 레벨 + 집중도 기반 ETF 매칭 스코어링

## 4. 프론트 UX 반영
1. 대시보드에 3개 신규 카드 추가
- AI 포트폴리오 진단
- 리밸런싱 권장 목록
- 추천 ETF 포트폴리오

2. UI 디테일
- 지표별 색상 시맨틱(위험/주의/안정)
- 종목/ETF 추천 사유를 문장형으로 노출
- 실제 투자 실행 전 참고용 고지 문구 반영

## 5. 다음 확장 (실서비스 단계)
1. 실제 LLM 연동
- 프롬프트: 지표 + 계좌 맥락 + 사용자 목표수익률/손실한도
- 출력: 실행 가능한 액션 플랜 + 설명가능성 근거

2. 개인화 강화
- 사용자 투자성향 설문 기반 목표 비중 엔진
- 세금/수수료/슬리피지 반영 리밸런싱 비용 최소화

3. 운영 고도화
- 어드바이저 배치 스케줄 + 알림 자동 발송
- 추천 성과 추적(추천 당시 대비 성과) 및 모델 피드백 루프
