# 기능 카탈로그 (현행 vs 목표)

## 1. 현행 기능 (As-Is)
- 데모/헬스
  - `/hello`
- 사용자
  - 단건 조회(`/findOne`)
- 종목
  - 종목 목록 조회(`/findAll`)
  - 임시 뉴스 제목 검색(`/stock/findAllTmpNews`)
  - 정적 평가값(`/stock/getStockEvaulation`)
- UI
  - RN 기본 레이아웃 + 검색 입력창 수준

## 2. 목표 기능 (To-Be)
### A. 계정/인증
- 이메일/소셜 가입/로그인
- 토큰 갱신 및 세션 제어
- 사용자 환경설정(시간대/언어/채널)

### B. 관심종목/탐색
- 멀티 워치리스트
- 종목 검색 + 별칭 검색
- 저장형 스크리너(시총/섹터/가격 액션)

### C. 포트폴리오
- 포트폴리오/계좌 생성
- 거래 원장(매수/매도/배당/수수료)
- 실시간 포지션, 실현/미실현 손익
- 자산 타임라인

### D. 시장/뉴스 인텔리전스
- 캔들 수집/정규화
- 기사 수집/중복 제거
- 기사-종목 매핑
- 감성 점수 및 추세

### E. 인사이트/알림
- 이상탐지
  - 급격한 가격 변동
  - 거래량 급증
  - 부정 뉴스 급증
- 사용자 규칙 기반 알림
- 근거 설명 가능한 시그널

### F. 알림/재참여
- 인앱 알림
- 이메일 발송 + 재시도 정책
- 읽음 상태 포함 알림함

### G. 운영/관리
- 수집 잡 상태/재실행
- 소스 품질/지연 모니터링
- 중요 행위 감사로그

## 3. 핵심 사용자 여정
- 여정 1
  - 회원가입 -> 워치리스트 생성 -> 종목 추가 -> 일간 요약 확인
- 여정 2
  - 포트폴리오 생성 -> 거래 입력 -> 손익 모니터링 -> 알림 수신
- 여정 3
  - 알림 진입 -> 근거 뉴스/시세 확인 -> 행동 결정
